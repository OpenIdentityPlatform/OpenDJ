How To Configure OpenDJ

If you are reading this page, you have already installed the OpenDJ package.

Before You Configure

A Java SE 6.0 (version 1.6.0_10) or higher runtime environment must be installed. It is recommended that your system have at least 100MB of disk space and 256MB of free memory to run OpenDJ for evaluation purposes. Note that you can configure OpenDJ in such a way that it requires substantially less, or more, disk space. For more information, see the System Requirements section of the OpenDJ Installation Guide.

To Configure OpenDJ

  1. As root, type the following command:
    $BASEDIR/opendj/configure --instancePath {instancePath} [ --userName {userName} [ --groupName {groupName} ] ]
  2. As {userName}, type the following command:
    $BASEDIR/opendj/setup
