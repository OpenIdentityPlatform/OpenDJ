Test ID windows_service
Panel
 OpenDS and windows services: run the GUI setup and enable Windows Service for Opends, Uninstall with the GUI and disable Windows Service.
Items
 - install OpenDS using the same procedure than in the setup_gui_ssl testcase (GUI with security enabled, status panel)

- on the last panel, select "Run OpenDS as a Windows Service"

- Stop OpendDS server using the Services Manager

- Start OpendDS server using the Services Manager

- Stop OpendDS server using the Services Manager

- Restart the host and verify that opends is started after reboot

- Stop OpendDS server using the Services Manager

- run setup command to uninstall in GUI, and select "Unregister OpenDS as a Windows service"
Specific checks - Check the install is successfull
and the opends service is listed and showing correct status

- Check status-panel data are valid

- Check uninstall is successfull
and the opends service is not anymore listed
